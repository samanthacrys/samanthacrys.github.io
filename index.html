Welcome to my ePortfolio for my CS 499 capstone. I am Samantha Durr, a student of computer science. My past history is that of a published author. I have a couple of degrees already and this is my fourth degree. My focus has been in software engineering and the following components within my portfolio are the three projects in software design and engineering, algorithms and data structures, and databases. I focused on creating complete projects and also security within these. The following links to the narrative stories of each project.

Artifact 1

Artifact 1 is the Software Design and Engineering project. This is the narrative of what took place during the enhancement process. This project was a space based text-adventure that was written in Python and with minimal design elements. The enhancements added functionality and it was converted to a more common programming language with more capabilities.

Artifact 2

Artifact 2 is the Algorithms and Data Structures project. This is the narrative that took place when working on the project. This project was a Task class where you can add new tasks to a dictionary and have tests that makes sure that all of the methods work as they are supposed to. My enhancements involved reducing redundancy and converting it to a more secure programming language.

Artifact 3

Artifact 3 is the Database project. This narrative explains my process of working on the project. The original project was to create a database that could connect to a MongoDB system and access the data using a web-based dashboard that allowed for specific sorting capabilities. My enhancments ended up being related to security and authentication.
