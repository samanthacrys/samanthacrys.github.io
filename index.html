Welcome to the Interactive Fiction game, Defeat the Glabgargs. This site is a github pages site that walks through the process of creating this game, while also allowing access for playing the game in general. Throughout this journey, there are multiple versions of thei game that shows you how to create a text adventure and an interactive fiction through multiple types or processes. This even goes through the history of creating two of these versions of the program and also designing a simple game. You will be provided with explanations, examples, walk-throughs, and the history behind the versions as well. All of this will hopefully help you if you're interested in making a text-adventure. These different sections are there to help people new to programming, new to game development, and those that want to learn different development environments. Hope you enjoy what you see.

All main sections are accessible via the navigation bar. When new articles are posted, there will be updated information in the section articles on this page that will have the direct links to those articles.
