Welcome to the Inky version of Defeat the Glabgargs. This is your step up from Python. While Python has you building the entire game from here, it is still very basic with minimal elements. What you are getting by moving to Inky is to pull yourself away from the coding to see how you can improve the game to make it into an interactive fiction. Inky is a non-coding development engine. You can use it to help you learn how to build up a game that has flags, different dialog based on your interactions and basically make a stronger format for a choose your own adventure. Instead of just being a basic grab this item, move here, and so on that you see the Python version. This is basically your transition before you get into more coding and more work of Godot and C#.
